# Metric / Imperial Unit Converter

Simple Chrome Extension for converting:

- Meters ↔ Feet
- Liters ↔ Gallons
- Kilograms ↔ Pounds

Built with:

- HTML
- CSS
- JavaScript

---

# Preview

The extension allows you to enter a number and instantly convert units between metric and imperial systems.
